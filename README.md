# Autonomous Robot Car (Arduino)

This repository contains the design and implementation of an autonomous robot car using Arduino. The system integrates multiple sensors and control algorithms to achieve line following, distance measurement, obstacle detection, and orientation sensing.

## Project Objectives
- Build an autonomous mobile robot
- Measure travelled distance using a rotary encoder
- Detect obstacles using an ultrasonic sensor
- Sense orientation using an MPU-6050 IMU
- Improve line-following performance using PID control

## System Features
- Line following using IR sensors with PID control
- Distance measurement using encoder interrupts
- Obstacle detection using HC-SR04 ultrasonic sensor
- Orientation sensing using MPU-6050 accelerometer and gyroscope
- Real-time display using a 16×2 LCD

## Hardware Components
- Arduino Uno  
- L298N Motor Driver  
- DC Motors with Encoder  
- IR Line Sensors  
- HC-SR04 Ultrasonic Sensor  
- MPU-6050 
- 16×2 LCD Display
